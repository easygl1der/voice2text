# Voice2Text - 语音输入应用设计

## 1. 项目概述

**项目名称**: Voice2Text
**类型**: macOS 菜单栏应用 (Menu Bar App)
**核心功能**: 全局快捷键触发语音录制，自动转写后输入到当前光标位置，同时保存转写历史

## 2. 用户场景

- 用户在任何应用中工作时，按下快捷键说话
- 松开快捷键后，语音被发送到 API 转写
- 转写结果自动粘贴到当前光标位置
- 所有转写内容保存到本地历史记录

## 3. 技术栈

- **框架**: SwiftUI + AppKit
- **音频**: AVFoundation
- **网络**: URLSession
- **存储**: UserDefaults + SQLite
- **快捷键**: HotKey library (或 Carbon APIs)

## 4. 支持的转写服务

| 服务 | 特点 | API 格式 |
|------|------|----------|
| Deepgram | 低延迟、高准确率 | REST |
| AssemblyAI | 说话人分离 | REST |
| Mistral | 多语言支持 | REST |
| 本地 Whisper | 离线可用 | local |

## 6. 核心功能

### 6.1 全局快捷键
- 默认: ⌥ Option + V (可自定义)
- 按下开始录音，松开停止录音
- 需要 Accessibility 权限

### 6.2 录音与转写
- 使用 AVFoundation 录制音频
- 支持在线 API (Deepgram/AssemblyAI/Mistral)
- 离线模式使用本地 Whisper 模型

### 6.3 自动粘贴
- 使用 Clipboard 或模拟按键输入
- 支持 NSPasteboard + CGEvent 模拟粘贴

### 6.4 历史记录
- SQLite 本地存储
- 支持搜索、导出、删除

### 6.5 菜单栏界面
- 状态指示 (就绪/录音中/转写中)
- API 选择、语言选择
- 快速访问历史记录
